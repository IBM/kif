class Filter(subject: TFingerprint | None = None, property: TFingerprint | None = None, value: TFingerprint | None = None, snak_mask: TSnakMask | None = None, subject_mask: TDatatypeMask | None = None, property_mask: TDatatypeMask | None = None, value_mask: TDatatypeMask | None = None, rank_mask: TRankMask | None = None, language: TTextLanguage | None = None, annotated: bool | None = None)[source]ÔÉÅ

Bases: KIF_Object

Criterion for matching statements.

A statement matches a filter if:

  • Its subject matches the filter subject.

  • Its property matches the filter property.

  • Its value (if any) matches the filter value.

  • Its snak type matches the filter snak_mask.

  • Its subject type matches the filter subject_mask.

  • Its property type matches the filter property_mask

  • Its value type (if any) matches the filter value_mask.

  • Its rank (if any) matches the filter rank_mask.

  • Its value language (if any) matches the filter language.

Parameters:

  • subject ‚Äì Subject fingerprint.

  • property ‚Äì Property fingerprint.

  • value ‚Äì Value fingerprint.

  • snak_mask ‚Äì Snak mask.
